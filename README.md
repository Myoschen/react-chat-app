# react-chat-app

> Build a realtime chat app with React, Socket.io, MongoDB Atlas.

## Demo

More to come...

## How to run

> Download this project to your PC.

- Server side

  ```ps
  npm ci
  npm run start 
  ```

- Client side

  ```ps
  npm ci
  npm run start 
  ```
